# ◻️ Remove branding

You can remove branding on any paid account for $49/mo, this feature is also included as part of the Enterprise package.&#x20;

To remove branding just select the remove branding option from _**AI agent setup > Customize**_ on your Dashboard and the "Powered by My AskAI" text will disappear.
